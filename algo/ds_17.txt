#implement bubble sort

1. start
2. read size of the array and the read items for the arry using forloop
3. bubble sort
4. Starting with the first element(index = 0), compare the current element with the next element of the array.
5. If the current element is greater than the next element of the array, swap them
6. If the current element is less than the next element, move to the next element. Repeat Step 4.
7. stop
